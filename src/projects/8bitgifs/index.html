{% markdown %} Intrigued by the amazing gifs of [IWDRM]("http://iwdrm.tumblr.com/"), I thought it'd be fun to try applying some effects to the creator's already-mesmerizing cinemagraphs. The effect is an augmented version of nearest neighbor applied to each frame. The gif's colors are first analyzed, and the top ten are pulled out. The pixelate effect is then applied to each frame, limiting the colors to the pre-determined set, resulting in a much move "video-gamey" effect than if a standard nearest-neighbor were applied to each frame in isolation. {% endmarkdown %}
